# soccer-koa
Simple REST api which is written using Node.js, Koa, Typescript, Sequelize.

Entities are Country and Team.

## Country
You can:
- get countries list
- get a country by id
- add, edit, remove country by id

## Team 
You can
- get teams list
- get teams by countryId
- get a team by id
- add, edit, remove team by id
